Funciones rápidas de teclado

Puede utilizar su teclado para llevar a cabo rápidamente muchas acciones en Vienna. Para encontrar las combinaciones de teclas para las acciones más frecuentes, consulte en los menús o bien desplácese hacia el final de esta página. Recuerde que muchos ítems en Vienna disponen de menú contextual. Para acceder a ellos, pulse la tecla Control y haga clic sobre el ítem seleccionado.

Para llevar a cabo una acción, pulse las siguientes combinaciones de teclado.

Tecla Acción
Funciones rápidas de teclado  
f Abre la barra de filtros y se enfoca en el campo de búsqueda del filtro.
h Desplaza el cursor al cajón de búsqueda.
k Marca la carpeta seleccionada como leída.
m Marca con indicador el artículo seleccionado.
b Muestra el primer artículo no leído.
n Muestra el siguiente artículo no leído.
r or u Marca el artículo seleccionado como leído.
s Marca todo el contenido de la carpeta seleccionada como leído y salta a la siguiente carpeta con artículos no leídos.
< Muestra el artículo o página web previa.
> Muestra el siguiente artículo o página web.
Izquierda Mueve el enfoque a la lista de carpetas si actualmente está en la lista de artículos.
Barra espaciadora Muestra el siguiente artículo no leído. En el caso de que el artículo seleccionado disponga de más de una página, se mostrará la siguiente página del artículo.
Aceptar Abre la página original del artículo seleccionado.
Suprimir Envía el artículo seleccionado a la papelera. Si esta función se utiliza dentro de la papelera, el artículo se eliminará de manera permantente.
Funciones rápidas de teclado generales  
Alt-Click Abre el enlace clicado, ignorando tu preferencia por abrir enlaces en un navegador externo.
Shift-Comando-B Abre la barra de filtros.
Comando-N Crea una nueva suscripción.
Shift-Comando-G Crea un grupo de carpetas.
Shift-Comando-F Crea una nueva carpeta inteligente.
Shift-Comando-E Edita la dirección URL de la suscripción seleccionada o los criterios de una carpeta inteligente.
Shift-Comando-D Elimina la carpeta seleccionada.
Comando-C Copia el texto seleccionado al portapapeles. Si el enfoque está en un artículo, copia los detalles del artículo al portapapeles.
Comando-H Oculta la ventana principal de Vienna.
Alt-Comando-H Oculta todas las demás ventanas de la aplicación, pero mantiene visible la ventana principal de Vienna.
Comando-I Muestra la ventana de información para la carpeta seleccionada.
Comando-L Coloca el enfoque en la barra de direcciones de la pestaña actual. O, si no hay un navegador abierto y enfocado, abre una nueva pestaña y coloca el enfoque en la barra de direcciones.
Shift-Comando-N Renombra la carpeta seleccionada.
Comando-R Actualiza todas las suscripciones.
Shift-Comando-R Actualiza únicamente las suscripciones seleccionadas.
Control-Comando-S Cancela cualquier actualización que se esté produciendo.
Comando-P Imprime el artículo seleccionado.
Shift-Comando-P Muestra la configuración de página.
Comando-Q Salir de Vienna.
Comando-T Abre una nueva pestaña y coloca el enfoque en la barra de direcciones.
Comando-U Muestra el siguiente artículo no leído.
Comando-Z Deshace la última acción, si es posible.
Comando-/ Activa o desactiva la barra de estado.
Comando-+ Aumenta el tamaño del texto en el artículo o página web actual.
Comando-Minus Disminuye el tamaño del texto en el artículo o página web actual.
Comando-, Abre el panel de preferencias de Vienna.
Alt-Command-Backspace Vacía la carpeta de papelera después de solicitar confirmación.
Comando-? Muestra la ayuda de Vienna.
Funciones rápidas de teclado para artículos  
Alt-Enter/Alt-Return Abre la página web original del artículo seleccionado, ignorando tu preferencia por abrir enlaces en un navegador externo.
Shift-Comando-Z Rehace la última acción, si fue deshecha anteriormente.
Shift-Comando-M Marca con indicador los artículos seleccionados.
Shift-Comando-U Marca como leídos los artículos seleccionados.
Alt-Comando-S Utiliza tu aplicación de correo predeterminada para enviar un enlace al artículo actual por correo electrónico.
Shift-Comando-S Marca como leída la carpeta seleccionada y se desplaza a la siguiente carpeta con artículos no leídos.
Shift-Comando-O Restaura un artículo desde la papelera a su carpeta original.
Shift-Comando-K Marca como leídos todos los artículos de las carpetas seleccionadas.
Alt-Comando-K Marca como leídos todos los artículos de todas las carpetas.
Funciones rápidas de teclado para ventanas  
Comando-W Cierra la página activa en el caso de que haya alguna. De no haber ninguna, cierra la propia ventana principal de Vienna.
Alt-Comando-W Cierra todas las pestañas. Tenga en cuenta que que esta acción sustituye al comamndo Cerrar pestaña del menú Archivo.
Control-Comando-Izquierda Muestra la pestaña anterior.
Control-Comando-Derecha Muestra la pestaña siguiente.
Comando-Y Abre la pestaña de la lista de artículos.
Comando-M Minimiza la ventana de Vienna en el dock.
Comando-0 Muestra la ventana de actividad.
Comando-1 Muestra la ventana principal de Vienna.
Comando-2 Muestra la ventana de descargas.
Funciones rápidas de teclas de ventana  
Alt-R Actualiza la página web seleccionada.
Comando-Period Cancela la carga de la página web seleccionada.
Comando-[/Comando-Izquierda Muestra la anterior página web seleccionada.
Comando-]/Comando-Derecha Muestra la siguiente página web seleccionada.
Comando-F Desplaza el cursor al cajón de búsqueda. Escriba alguna palabra y/o texto, pulse la tecla Aceptar y Vienna buscará en la página web seleccionada su solicitud.
Comando-G Busca la siguiente acepción del cajón de búsqueda en la página web.
