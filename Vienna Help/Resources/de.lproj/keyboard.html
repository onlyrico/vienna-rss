Tastaturkurzbefehle

Du kannst mit deiner Tastatur viele Aufgaben in Vienna schnell erledigen. Die Tastenkombinationen für häufig verwendete Befehle findest du in den Menüs in der Menüleiste sowie in der Liste unten. Viele Elemente in Vienna, wie beispielsweise der Ordnerbereich und der Artikellistenbereich, verfügen auch über Kontextmenüs, die durch einen Rechtsklick auf das Element beziehungsweise durch das Drücken und Halten der Control-Taste und einen Linksklick auf das Element angezeigt werden können.

Um eine Aufgabe auszuführen, drücke die unten aufgeführten Kurzbefehle.

Kurzbefehl Aktion
Einzeltastenkurzbefehle  
f Öffnet die Filterleiste und setzt den Eingabefokus auf ihr Suchfeld.
h Legt den Eingabefokus auf das Suchfeld.
k Markiert den aktuellen Ordner als gelesen.
m Markiert die ausgewählten Artikel.
b Zum ersten ungelesenen Artikel gehen.
n Zum nächsten ungelesenen Artikel gehen.
r oder u Markiert die ausgewählten Artikel also gelesen.
s Markiert den aktuellen Ordner als gelesen und springt zum nächstne.
< oder Komma (,) Zeigt den vorigen Artikel oder die vorige Webseite.
> oder Punkt (.) Zeigt den nächsten Artikel oder die nächste Webseite.
Linkspfeil Setzt den Fokus auf die Ordnerliste, falls er sich aktuell auf der Artikelliste befindet.
Leertaste Springt zum nächsten ungelesen Artikel. Falls der momentane Artikel noch nicht bis zum Ende gescrollt ist, springt die Anzeige des momentanen Artikels um eine Seite.
Zeilenschalter Öffnet die permanente Webadresse des momentanen Artikels.
Löschen Verschiebt die ausgewählten Artikel in den Papierkorb. Wenn die Taste im Papierkorb gedrückt wird, werden alle darin liegenden Artikel permanente gelöscht.
Allgemeine Kurzbefehle  
Wahltaste-Klick Öffnet den angeklickten Link, ungeachtet der Einstellung, ob Links im externen Browser geöffnet werden sollen.
Umschalt-Befehl-B Öffnet die Filterleiste.
Befehl-N Ein neues Abonnement erstellen.
Umschalt-Befehl-G Erstellt einen Gruppenordner.
Umschalt-Befehl-F Erstellt einen intelligenten Ordner.
Umschalt-Befehl-E Bearbeitet die URL oder die Filterkriterien des gewählten Ordners.
Umschalt-Befehl-D Löscht den ausgewählten Ordner.
Befehl-C Copies the selected text to the clipboard. If the focus is on an article, it copies the article details to the clipboard.
Befehl-H Hides the Vienna main window.
Wahl-Befehl-H Hides all other application windows but keeps the Vienna main window visible.
Befehl-I Displays the Info window for the selected folder.
Befehl-L Puts the focus on the address bar of the current tab. Or if no browser is open and has the focus, opens a new tab and puts the focus on the address bar.
Umschalt-Befehl-N Rename the selected folder.
Befehl-R Alle Abonnements aktualisieren.
Umschalt-Befehl-R Nur die ausgewählten Abonnements aktualisieren.
ctrl-Befehl-S den momentanen Aktualisierungsvorgang abbrechen.
Befehl-P Die ausgewählten Artikel drucken.
Umschalt-Befehl-P Seite einrichten.
Befehl-Q Exit Vienna.
Befehl-T Opens a new tab and puts the focus on the address bar.
Befehl-U Springt zum nächsten ungelesen Artikel.
Befehl-Z Undoes the last action, if it can be undone.
Befehl-Schrägstrich (/) Toggles the status bar on or off.
Befehl-Plus-Zeichen (+) Increases the size of the text in the current article or web page.
Befehl-Minus-Zeichen (–) Decreases the size of the text in the current article or web page.
Befehl-Komma (,) Opens Vienna settings.
Wahl-Befehl-Rückschritt Empties the trash folder after prompting for confirmation.
Befehl-Fragezeichen (?) Aufrufen der Vienna-Hilfe.
Artikelkurzbefehle  
Wahl-Zeilenschalter Opens the selected article's original web page, overriding your setting for opening links in external browser.
Umschalt-Befehl-Z Redo the last action, if it was previously undone.
Umschalt-Befehl-M Markiert die ausgewählten Artikel.
Umschalt-Befehl-U Markiert die ausgewählten Artikel als gelesen.
Wahl-Befehl-S Uses your default mail application to send a link to the current article by e-mail.
Umschalt-Befehl-S Marks the current folder read then skips to the next folder with unread articles.
Umschalt-Befehl-O Restores an article from the trash back to its original folder.
Umschalt-Befehl-K Markiert alle ausgewählten Artikel als gelesen.
Wahl-Befehl-K Marks all articles in all folders read.
Fensterkurzbefehle  
Befehl-W Schließt den aktuellen Tab. Falls kein Tab aktiv ist, wird das Hauptfenster geschlossen.
Wahl-Befehl-W Schließt alle Tabs.
Steuerung-Befehl-Linke Pfeiltaste Springt zum vorigen Tab.
ctrl-Befehl-Rechtspfeil Springt zum nächsten Tab.
Befehl-Y Öffnet die Artikelliste.
Befehl-M Minimiert Vienna in das Dock.
Befehl-0 Aktiviert die Aktivitätenanzeige.
Befehl-1 Öffnet das Vienna Hauptfenster falls es zuvor geschlossen wurde.
Befehl-2 Aktiviert das "Geladene Dateien"-Fenster.
Browserkurzbefehle  
Befehl-R Aktualisiert die momentanen Webseite.
Befehl-Punkt (.) Bricht das laden der aktuellen Webseite ab.
Befehl-linke eckige Klammer ([) Springt zur vorigen Webseite.
Befehl-rechte eckige Klammer (]) Springt zur nächsten Webseite.
Befehl-F Legt den Eingabefokus auf das Suchfeld. Durch drücken von Enter wird auf der momentanen Webseite nach dem eingegebenen Begriff gesucht.
Befehl-G Sucht bis zum nächsten Vorkommen des Suchbegriffes auf der Webseite.
