Réglages Avancés

Le panneau Avancés des Réglages propose des options pour changer des réglages très spécifiques de Vienna qui ne sont pas essentiels pour une utilisation classique, mais qui peuvent résoudre des problèmes comme ceux traités par le forum de Support. Cette page détaille les réglages avancés et les fonctions.

Activer JavaScript dans le navigateur interne

Active ou désactive le support de script JavaScript dans le navigateur interne de Vienna. Est activé par défaut. En décochant ce réglage, le code JavaScript des pages Web ne sera pas exécuté. Cela peut résoudre des problèmes avec les pages qui utilisent des scripts pour afficher des fenêtres ou d'autre comportement inattendus, mais cela peut aussi rendre l'affichage de page incomplet ou créer des dysfonctionnements.
