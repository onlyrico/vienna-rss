Avancerade inställningar i Vienna

Avancerade inställningar ger mycket specifika alternativ som inte är nödvändiga för normal användning men som kan lösa vissa problem som rekommenderas av supportforumet. Den här sidan beskriver de avancerade inställningarna och deras funktion.

Aktivera JavaScript i inbyggda webbläsaren

Aktiverar eller inaktiverar stöd för JavaScript i Viennas interna webbläsare. Som standard är detta aktiverat. Genom att stänga av den här inställningen tillåts inte JavaScript-kod på webbsidor att köras. Detta kan lösa problem med sidor som använder skript för popup-fönster eller annat oväntat beteende, men om du stänger av inställningen kan det också leda till att vissa webbsidor verkar ofullständiga eller inte fungerar korrekt.
